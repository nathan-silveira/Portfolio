Análise de Dados - Steam
- Descrição do projeto: Criação de visualização interativa para explorar lançamentos, avaliações e reviews de jogos presentes na plataforma Steam
- Habilidades demonstradas: Coleta, preparação e análise de dados para obter insights valiosos sobre o mercado de jogos. Habilidades em design de visualização e uso do Power BI
- Link para visualização: Power BI.
Análise de Dados - Chicago: Crimes
- Descrição do projeto: Desenvolvimento de um dashboard, no Power BI, para analisar dados de crimes ocorridos em Chicago de 2001 até 2023
- Habilidades demonstradas: Demonstração de habilidades em coleta, preparação e análise de dados, manipulação de dados complexos, criação de medidas e métricas e desenvolvimento de filtros interativos
- Link para visualização: Power BI.
Análise de Dados - Bloco de código
- Descrição do projeto: Desenvolvimento de uma análise abrangente dos dados da campanha de marketing da empresa fictícia "Bloco de Código".
- Habilidades demonstradas: Neste projeto, foram aplicadas habilidades em coleta e análise de dados de marketing, assim como a utilização de recursos do Power BI para criar visualizações significativas e interativas. A interpretação dos dados e utilzação de métricas como CTR (Click-Through Rate), permitiu obter insights valiosos sobre o desempenho da campanha de marketing, as preferências do público-alvo e o crescimento da base de usuários
- Link para visualização: Power BI.
Análise de Dados - Hermes
- Descrição do projeto: Realização de uma análise detalhada dos dados logísticos de entregas da empresa fictícia Hermex
- Habilidades demonstradas: foram aplicadas habilidades em coleta e análise de dados logísticos, além do uso avançado do Power BI para criação de visualizações interativas e informativas. A análise dos dados logísticos, utilizando métricas como STD (Ship To Door), permitiu identificar padrões, tendências e áreas de melhoria no processo de entregas da empresa fictícia "Hermex"
- Link para visualização: Power BI.
